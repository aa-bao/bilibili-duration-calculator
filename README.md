# BiliBilli视频分集总时长计算插件

google浏览器使用：

进入：`chrome://extensions/`

打开开发者模式，然后加载已解压的扩展程序，选中此插件目录 `bilibili-duration-calculator` 即可

edge同样，进入：`edge://extensions/`

打开开发者模式，然后加载已解压的扩展程序，选中此插件目录 `bilibili-duration-calculator `
